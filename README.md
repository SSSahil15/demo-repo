# 🚀 DevPulse Demo Repository

This repository is designed to demonstrate DevPulse features:

- CI/CD pipeline behavior
- Failing & passing tests
- Security vulnerabilities
- Docker builds
- AI Copilot insights
- DevPulse Score changes

## 🧪 Scenarios to Test

### ❌ Scenario 1: Failing Pipeline
- test_fail() is active
- CI/CD will show failure

### ✅ Scenario 2: Fix Tests
Change failing test to pass and push again.

### 🔐 Scenario 3: Vulnerabilities Present
Outdated dependencies trigger security scans.

### 🟢 Scenario 4: Fix Vulnerabilities
Update dependencies to secure versions and push again.

## 🎯 Purpose
Demonstrate DevOps + DevSecOps + AI workflows using DevPulse.
